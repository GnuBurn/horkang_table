# Horkang Table (Smart Library Monitoring)

This is a term-project for scanning the table status using Computer vision
# Disclaimer
Due to the copyright, We cannot share the data to public.

## Initialize
1. Install either Anaconda or Minicoda.
2. Go to the project path.
3. Create a environment using
"conda env create --file environment.yml".
4. Activate the dependencies using
"conda activate cctv_env".
5. Don't forget to change the interpreter.
